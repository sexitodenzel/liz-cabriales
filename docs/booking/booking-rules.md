
## Contexto previo

### Requisitos para reservar
- Se requiere cuenta activa para poder agendar una cita
- El pago es 100% adelantado — sin anticipo parcial, se paga el total al momento de reservar

- El cliente puede cancelar su cita con mínimo **1 día de anticipación**
- Si el cliente no cancela con tiempo o no se presenta, **pierde el dinero** (sin reembolso)

### Restricciones
- Un usuario solo puede tener **1 cita reservada a la vez**
